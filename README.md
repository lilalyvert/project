# Stress Tracker App

A simple browser-based stress tracking application built using HTML, CSS, and vanilla JavaScript.

## Features
- Input stress data (stressor, severity, coping strategies)
- Automatic stress classification
- User-friendly summary output
- Researcher-facing structured record
- LocalStorage support (optional)

## How to Run
Open `stress-tracker.html` in any browser.

## Project Structure
- agent.md → agent rules and constraints  
- HARNESS.md → how the app runs and is tested  
- plan.md → project plan  
- stress-tracker.html → main application
